### 1. 🧪 赛博酸性配色方案 (Acid & Cyber Palette)

这套配色的核心是**“极致的黑”与“极致的亮”**的冲撞。我们要利用OLED屏幕的特性，展现出一种带电的视觉张力。

|**颜色名称**|**Hex Code**|**使用场景说明**|**视觉心理学**|
|---|---|---|---|
|**Void Black (虚空黑)**|`#050505`|**全局背景**。比普通黑色更深邃，仿佛手机屏幕被挖了一个洞。|营造无限的空间感，突出前景的光效。|
|**Acid Green (酸性绿)**|`#CCFF00`|**高亮/关键按钮**。极高饱和度的荧光绿，Y3K风格的标志性颜色。|视觉冲击力极强，代表“能量”与“信号”。|
|**Liquid Chrome (液态铬)**|`#E6E8EA`|**硬币/金属材质**。带有高反射率的银白色，模拟流动的金属水银。|赋予物体一种“未来的奢侈感”。|
|**Electric Purple (电光紫)**|`#BC13FE`|**辅助光效/渐变**。用于背景的弥散光（丁达尔效应）或故障风效果。|增加神秘感和赛博朋克氛围。|
|**Glitch Blue (故障蓝)**|`#00F0FF`|**数据/边框**。经典的科技蓝，用于装饰性的HUD（抬头显示）元素。|传达“数字化”和“精准”的信号。|

### 2. 👽 字体排印建议 (Typography)

这个主题需要一种**“不稳定的秩序感”**。我们将使用“酸性设计”中常见的宽体字和像素字进行混排。

- **主要字体 (标题/神谕): Extended Sans (宽无衬线体)**
    
    - **推荐字体：** _Syne_ (极具艺术感的宽体字) 或 _Unbounded_。
        
    - **设计意图：** 这种字体看起来像是被横向拉伸过，带有一种扭曲的张力，非常符合Y3K的审美。
        
- **功能字体 (数据/代码): Pixel / Glitch (像素/故障体)**
    
    - **推荐字体：** _VT323_ 或 _Rubik Glitch_。
        
    - **设计意图：** 用于显示“Loading...”、“Connecting...”等微文案，模拟老式计算机或黑客终端的视觉语言。

布局不再遵循传统的从上到下，而是采用**中心辐射式**。周围布满了装饰性的十字准星、条形码和无意义的数据流。

### 4. ⚡️ 关键交互细节 (UX Micro-interactions)

这里的交互核心是**“充能”与“释放”**。我们不模仿物理抛物线，我们模仿“数据传输”。

1. **待机状态 (Idle)：**
    
    - 那团“液态金属”会像呼吸一样缓慢蠕动，表面反射着周围（假）环境的光影（模拟光追效果）。背景中有微弱的噪点在流动。
        
2. **操作过程 (Action)：**
    
    - **动作：** **长按屏幕**。
        
    - **视觉：** 随着按住的时间，屏幕周围出现向中心汇聚的电流线条（电光紫/酸性绿）。手机开始高频微震（Taptic Engine 模拟电流过载）。
        
    - **释放：** 松开手指瞬间，屏幕出现瞬间的**故障效果 (Glitch)**——画面撕裂、色差偏移（RGB Split）。
        
3. **结果揭晓 (Reveal)：**
    
    - 故障闪烁结束后，液态金属瞬间凝固成一枚表面光滑、带有浮雕图案的未来硬币，定格在屏幕中央。
        
    - 结果文字（YES/NO）不是直接显示，而是像黑客电影里破解密码一样，字符快速跳动最后停在正确单词上。
        
4. **声音设计 (Sound)：**
    
    - **科幻感：**
        
        - 长按声：能量积蓄的升调音 (_Wooooop..._)。
            
        - 释放声：电流短路声 (_Zzzzt!_) + 重低音冲击 (_Boom_).
            
        - 背景音：深邃的宇宙嗡嗡声 (Drone ambient)。
            

### 5. 📸 专属小红书运营建议 (Bonus Strategy)

对于这个主题，你的“出片”策略是**“我的手机里住了个AI”**。

- **透明小组件 (The Widget):**
    
    - 一定要做一个**全透明背景**的小组件。只显示那个液态金属球和一句绿色的像素短语（如 `STATUS: LUCKY`）。
        
    - 当用户把它放在黑色或深色赛博风壁纸上时，看起来就像手机自带的高科技插件，这是小红书“桌面美学”区的顶流配置。
        
- **分享文案：**
    
    - 生成的图片要带有一个巨大的、像警告标志一样的二维码。文案设计成：“宇宙信号已接收 / Signal Received”。
