# Flip Moment - 功能说明文档

## 应用简介

Flip Moment 是一款创新的决策辅助应用，通过富有创意的动画交互帮助用户做出随机决策。应用采用多主题皮肤系统，为用户提供四种不同的视觉体验：复古机械感、治愈柔和风、赛博科技感和许愿梦幻风。

## 核心功能

### 🎯 智能决策系统

#### 功能描述
- **一键决策**: 用户只需点击屏幕，即可获得随机的二元决策结果（是/否）
- **多场景适用**: 适用于日常选择、决策辅助、趣味游戏等多种场景
- **即时反馈**: 提供即时的决策结果反馈

#### 技术实现
- 基于随机算法生成决策结果
- 与当前使用的主题皮肤深度集成
- 决策历史自动记录和保存

#### 用户体验
- 简单直观的操作方式
- 流畅的动画过渡效果
- 丰富的视觉和音效反馈

### 🎨 多主题皮肤系统

#### 复古风格（Vintage）
- **设计理念**: 模拟真实的3D硬币翻转体验
- **视觉效果**: 
  - 深色金属质感背景
  - 金黄色强调元素
  - 机械感强烈的硬币动画
- **交互体验**: 
  - 序列帧硬币翻转动画
  - 机械感音效
  - 触感震动反馈
- **适用场景**: 商务决策、重要选择、正式场合

#### 治愈风格（Healing）
- **设计理念**: 柔和舒缓的视觉体验
- **视觉效果**:
  - 米白色温暖背景
  - 温和的配色方案
  - Rive矢量动画
- **交互体验**:
  - 流畅的矢量动画
  - 柔和的音效
  - 舒缓的触感反馈
- **适用场景**: 日常选择、休闲时光、放松时刻

#### 赛博朋克风格（Cyber）
- **设计理念**: 未来科技感体验
- **视觉效果**:
  - 黑色高对比度背景
  - 霓虹色彩强调
  - 科技感UI元素
- **交互体验**:
  - 科技感动画特效
  - 电子音效
  - 未来感震动模式
- **适用场景**: 科技爱好者、未来感需求、特殊场合

#### 许愿风格（Wish）
- **设计理念**: 梦幻许愿体验
- **视觉效果**:
  - 渐变梦幻背景
  - 温暖梦幻色调
  - 飘逸的动画效果
- **交互体验**:
  - 梦幻粒子效果
  - 轻柔的音乐音效
  - 温和的触感反馈
- **适用场景**: 许愿祈福、浪漫时刻、特殊纪念日

### 📱 国际化支持

#### 语言支持
- **中文（简体）**: 完整的中文界面和交互
- **英文**: 英文界面和交互支持
- **动态切换**: 支持应用内语言实时切换

#### 本地化特性
- 皮肤名称和描述的本地化
- 完整的界面文本支持
- 文化适配的UI设计

### 📊 历史记录系统

#### 功能特性
- **自动记录**: 所有决策结果自动保存
- **详细信息**: 记录决策时间、结果、使用的皮肤模式
- **分类查看**: 按皮肤模式分类查看历史记录
- **本地存储**: 使用Hive数据库本地存储，数据安全可靠

#### 数据结构
```dart
{
  "id": "时间戳",
  "timestamp": "决策时间",
  "result": "决策结果",
  "skinModeName": "皮肤模式名称"
}
```

### 🎵 多感官反馈系统

#### 音效系统
- **皮肤专属音效**: 每种皮肤拥有独特的音效设计
  - 复古: 机械硬币翻转音
  - 治愈: 柔和自然音效
  - 赛博: 电子科技音效
  - 许愿: 梦幻轻柔音效
- **交互音效**: 点击、决策结果的音效反馈

#### 触感反馈
- **多级震动**: 不同操作对应不同强度的震动
- **皮肤适配**: 根据当前皮肤调整震动模式
- **硬件支持**: 充分利用设备硬件能力

#### 视觉效果
- **流畅动画**: 使用Flutter Animate实现丝滑动画
- **3D效果**: Vintage皮肤提供3D硬币翻转效果
- **粒子系统**: 各种特效和粒子效果

### 🏠 桌面组件支持

#### Home Widget功能
- **快捷决策**: 在桌面直接进行决策
- **主题同步**: 桌面组件与应用主题保持一致
- **实时更新**: 决策结果实时显示在桌面

### 🔧 设置系统

#### 个性化设置
- **主题切换**: 四种皮肤模式自由切换
- **音效控制**: 音效开关和音量调节
- **震动设置**: 震动反馈开关设置
- **语言设置**: 中英文切换

#### 数据管理
- **历史记录管理**: 查看、导出、清理历史记录
- **应用数据**: 存储空间使用情况
- **重置功能**: 应用数据重置选项

## 使用指南

### 🚀 快速开始
1. **启动应用**: 点击图标启动Flip Moment
2. **选择皮肤**: 根据喜好选择四种皮肤之一
3. **开始决策**: 点击屏幕中央的决策按钮
4. **查看结果**: 观看动画并获得决策结果
5. **查看历史**: 在历史页面查看所有决策记录
